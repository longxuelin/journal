## 6.配置页面布局说明
```bash
LLM 配置区域：
        1.API 密钥：输入 LLM 服务提供商的 API 密钥
        2.模型名称：选择使用的 LLM 模型
Telegram Bot 配置区域：
        1.Bot 令牌：输入从 BotFather 获取的 Telegram Bot 令牌
        2.Chat ID：输入接收通知的聊天 ID
验证选项：
        可选择验证 LLM 和 Telegram 配置的有效性
提交操作反馈
成功保存：
        显示绿色成功消息："配置已成功保存！"
        若勾选验证选项，会先验证配置有效性
        验证通过后会显示相应的成功提示
可能的错误提示：
        输入字段为空时显示红色错误消息
        验证失败时显示具体的错误原因（如 API 连接失败、Bot 令牌无效等）翻译成英文
